# Near v0.1.0

Near v0.1.0 is the first release of the backend-independent keyboard-first TUI platform and its Far-inspired reference file manager.

## Supported platforms

- macOS 13 or later on Apple Silicon and Intel through native builds.
- Current stable Ubuntu runners on x86-64.
- Current stable Windows runners on x86-64 with ConPTY support.

## Included applications

- `near-fm`: dual-panel Far-inspired file manager and full platform integration reference.
- `near-view`: standalone streaming text and hexadecimal viewer.
- `near-proc`: non-filesystem process-provider reference application.
- `near-demo`: mixed-provider surface and interaction laboratory.

## User-visible changes

- Semantic commands, contextual keymaps, themes, surfaces, dialogs, menus, help, tasks, and terminal lifecycle management.
- Provider-neutral navigation, search, streaming view, metadata, operations, handlers, macros, and embedded terminal support.
- Capability-controlled WebAssembly components and isolated process extensions.
- Reversible native path identity and platform adapters for macOS, Linux, and Windows.
- Patched Wasmtime 36.0 component runtime with a zero-advisory locked dependency audit at release time.

## Configuration and plugin migration

This is the first stable schema and protocol baseline. No configuration migration is required. Plugins must declare WIT package `near:plugin@0.1.0` or process protocol `near-process@0.1.0` and request explicit capabilities.

## Compatibility breaks

None from a previous stable release. APIs not listed as published in `docs/near-api-compatibility.md` remain internal.

## Known limitations

- Linux and Windows behavior is constrained by each platform's native metadata, Trash/Recycle Bin, shell, and terminal capabilities.
- Embedded PTY behavior depends on the installed native shell and terminal facilities.
- Process-extension sandbox enforcement is currently strongest on macOS; unsupported platforms fail closed where equivalent isolation is unavailable.

## Vulnerability exceptions

No vulnerability exception is active for this release.

## Rollback

Restore the previous application binaries and configuration directory backup. Near performs registered configuration migrations atomically and retains the last valid configuration when reload fails.

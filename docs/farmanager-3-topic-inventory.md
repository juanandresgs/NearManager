# Far Manager 3 Official Help Topic Inventory

This checklist maps the major English help topics in official source commit `28443e51d9f7b4b9e5976d1db8246532ccc40205`. It is intended as a coverage ledger for the broader research in `docs/farmanager-3-research.md`.

## Orientation

- `Contents` — master contents.
- `Help` — help navigation and search.
- `About` — product overview.
- `License` — revised BSD license.
- `CmdLine` — startup switches.
- `KeyRef` — keyboard-reference index.

## Keyboard reference

- `MenuCmd` — menu and list controls.
- `PanelCmd` — panel controls, views, selection, sorting, shortcuts, and path transfer.
- `PanelCmdSort` — sort-mode menu.
- `PanelSortCriteria` — detailed sort criteria.
- `FastFind` — incremental panel search.
- `CmdLineCmd` — command-line editing, completion, and history.
- `FuncCmd` — service and file-operation function keys.
- `DeleteFile` — delete, recycle, and wipe semantics.
- `Grabber` — console screen grabber.
- `MiscCmd` — common dialog, mouse, filtering, and macro controls.
- `SpecCmd` — service commands.
- `MsWheel` — wheel behavior by UI area.

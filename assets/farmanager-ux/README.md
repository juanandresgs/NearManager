# Far Manager UX Image Corpus

This directory is a source index for Far Manager interface research. It supports reconstruction of the application's interaction model, visual hierarchy, keyboard-first workflows, dialogs, panels, editor, viewer, plugins, and historical evolution without redistributing third-party media.

## Contents

- `manifest.csv` — per-item provenance, classification, source URL, and UX-state metadata.
- `../../docs/farmanager-3-visual-coverage.md` — workflow coverage matrix and known gaps.

## Interpretation

The manufacturer gallery is broad but visually represents an older Far generation. The `official-community` images are the strongest evidence for recent Far 3 details, but many are issue-focused rather than clean product shots. The NetBox guide is the clearest complete multi-step workflow in the corpus.

The filenames in the manifest are stable research identifiers. Retrieve media from its recorded source only when the source terms permit your intended use. Watermarks, issue annotations, localized strings, custom themes, and unusual console dimensions are part of the source context.

## Rights and Reuse

Every item retains a source page and source URL in `manifest.csv`. Wikimedia Commons items include their free-source context. No redistribution license is assumed for manufacturer-attributed, guide, gallery, or GitHub user-attachment media, so those media files are intentionally excluded from the public repository and release archives.
